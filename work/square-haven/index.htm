Square Haven

I cut my digital teeth on Square Haven. I learned digital technologies to express my interest in the videogames of the great RPG publisher, Squaresoft, and to create a meetingplace for friends with the same passion.

Preserved at squarehaven.com

Development

This site has a database of hundreds of games is maintained using PHP and MySQL. Each game has a hub page linking to relates modules -- news articles, videogame developers and creators, game guides, reader reviews, and more.

One of the main features is a series of comprehensive game guides written in HTML or using MySQL databases. These features bring in the most traffic from search engines, often from users searching for hints, items, or quests as they play the game. Some of our top entry points on Google are:

  • "hypello potion ffx": Square Haven Guides || Final Fantasy X items
  • secret of mana armor: Secret of Mana Guide - Armor - Square Haven
  • ff12 bazaar recipes: Final Fantasy XII Bazaar Goods List & Database -- Square Haven

In February 2020 I updated the backend PHP code to reflect the depreciation of the ancient mysql_... functions as of PHP 7. Because the functions were so extensively used throughout the code, I opted to replace it with mysqli_... functions. At the same time, I updated the code to better secure the backend from SQL injection by using prepare statements and mysqli_real_escape_string.
